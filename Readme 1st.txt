# TNB_billing_Calculation
Ini adalah projek untuk membuat kiraan kasar kos dan penggunaan tenaga elektrik. Namun terdapat kemungkinan bacaan adalah tidak sama seperti meter TNB atas beberapa faktor
  1. Faktor +- ketepatan sensor
  2. Faktor penghantaran data melalui network antara sensor dan reciever (wifi)
  3. Ketepatan waktu sebenar (HH:MM:SS) bacaan meter di ambil oleh meter reader atau smart meter
  
- Update akan dilakukan dari semasa ke semasa jika terdapat sebarang perubahan
  
Disclaimer : SEBARANG DATA YANG DI JANA ADALAH SEBAGAI RUJUKAN SAHAJA DAN TIDAK BOLEH DIGUNAKAN SEBAGAI DATA UNTUK TUNTUTAN SEBARANG GATI RUGI PADA BACAAN BILL TNB SEBENAR
